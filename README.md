# Hello Kitty Valentine Page

A playful “Will you be my Valentine?” page with a Hello Kitty-inspired theme, a funny share link, and a clickable moving heart.

## Files

- `index.html` – markup
- `styles.css` – theme styling
- `script.js` – heart motion + button reactions
- `breaking-news/index.html` – playful redirect link
- `system-check/index.html` – secret redirect link

## Run locally

Open `index.html` in a browser. That’s it.
